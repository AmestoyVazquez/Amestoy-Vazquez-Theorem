# Changelog

All notable changes to the Amestoy-Vázquez Conjecture project.

## [2.0.0] - Enero 2026

### Added
- Complete mathematical formulation of the conjecture
- Python implementation with verification system
- Unit tests for formula correctness
- MIT License for open source distribution
- Professional documentation and examples
- DOI citation (10.5281/zenodo.18248838)

### Mathematical Content
- Formula: S(n,d,a) = n·a + n(n^d - 1)/2
- Verification for dimensions 2-4
- Comparison with classical magic square formula
- Extension to arbitrary starting values

### Code Features
- Core formula implementation
- Magic square generation (odd n)
- Automated verification tests
- Usage examples

## [1.x] - Desarrollo inicial
- Preliminary research and formulation
- Early verification attempts
- Initial README documentation
